

RegExp:
   "--%WTPLSQL_set_dbout[(][[:alnum:]_$#]+[)]%--"

Example Annotation:
   --%WTPLSQL_set_dbout(SAMPLE_DB_OBJECT)%--

RegExp:
   "--%WTPLSQL_(begin|end)_ignore_lines%--"

Example Annotation:
   --%WTPLSQL_begin_ignore_lines%--


Run All:
 -) Loop through all Schema Defined for the SUT
    -) Load Test Runners into array
    -) Loop through Test Runners

Test Run:
 -) Create TEST_RUNS record
 -) Results Initialize
 -) Profiler Initialize
 -) Call Test Runner
 -) Profiler Finalize
 -) Results Finalize
 -) Update TEST_RUNS record

Results Initialize:
 -) Receive/Set TEST_RUN_ID
 -) Clear Results Array

Results Finalize:
 -) Copy Results Array to database tables.

Profiler Initialize:
 -) Receive/Set TEST_RUN_ID
 -) Clear Profiler Array
 -) Clear profiler_run_id
 -) Find DBOUT Annotations
 -) If DBOUT is not found, return
 -) Update TEST_RUNS
 -) Find Ignore Annotations and capture EXCLUDED_LINES array
 -) Start DBMS_PROFILER

Profiler Finalize:
 -) If profiler_run_id is clear, Return
 -) Stop DBMS_PROFILER
 -) DBMS_PROFILER.stop_profiler also copies data to DBMS_PROFILER tables.
 -) Remove non-DBOUT Lines from the DBMS_PROFILER tables
 -) Remove EXCLUDED_LINES from the DBMS_PROFILER tables
 -) Remove Non-Executable lines from the DBMS_PROFILER tables
 -) Copy results into DBOUT_PROFILE table.
 -) Copy Excluded Lines array to EXCLUDED_LINES table.
